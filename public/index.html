Features

Everything your agent needs.
Nothing it doesn't.

Built for power users who run their own infrastructure and want a proper mobile interface for it.

🔍

Real-time tool visibility

Watch every tool call as it happens — web search, code execution, file access — with live status badges and expandable debug traces.

✋

Human-in-the-loop approvals

Sensitive actions pause and route to you. Review the exact command, see the risk level, then approve or deny — before anything runs.

🔗

OAuth integrations

Connect Google, Microsoft, and other services to your agent in one tap. Tokens refresh automatically and never leave your device.

📱

Multi-connection profiles

Manage multiple OpenClaw servers from one app. Work server, home lab, cloud instance — switch profiles instantly.

💰

Token cost tracking

See live cost per message across Claude, GPT-4, and Gemini. Know exactly what each session costs before it adds up.

🖼

Image attachments

Send photos, screenshots, and camera captures directly to your agent for visual analysis, OCR, or code review.
